자주 묻는 질문
AppSuit AV는 어떤 OS와 개발 환경을 지원하나요?
+
Android와 iOS를 모두 지원하며, React Native·Flutter·Ionic 등 크로스플랫폼 환경에서도 동일하게 적용 가능합니다.
신규 OS 출시 시 사전 베타 테스트로 즉각 대응하여 최신 운영체제 호환성을 유지합니다.
금융보안원·행정안전부 규제 요건을 충족하나요?
+
전자금융감독규정(악성코드 감염 방지대책), 모바일 전자정부 서비스 관리 지침, 개인정보의 안전성 확보조치 기준 등 주요 규제의 모바일 백신 요구 항목을 모두 충족합니다.
도입 검토 시 공식 준수 문서를 제공해 드립니다.
SDK 적용 후 앱 성능에 영향을 주나요?
+
경량 SDK 방식으로 설계되어 앱 실행 속도 및 배터리 소모에 유의미한 영향이 없습니다.
패턴 업데이트는 백그라운드에서 자동 처리되며, 실제 고객사 적용에서도 사용자 경험에 영향을 주지 않음을 확인했습니다. 도입 전 POC를 통해 직접 검증하실 수 있습니다.
